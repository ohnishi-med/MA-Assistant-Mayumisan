### Phase 1: MVP (Minimum Viable Product) [完了]
**目標**: 電子化されたマニュアルの作成、閲覧、カテゴリによる多経路アクセスの基本機能を、ローカルアプリ(Electron)として実装する。

- [x] **1.1 プロジェクト環境構築**
- [x] **1.2 データベース設計・実装**
- [x] **1.3 カテゴリ管理機能 (IPC/UI)**

- [x] **1.5 複数ルート・詳細表示**

### Phase 2: 動的表示とメディア統合 [完了]
**目標**: アクセス経路に応じた動的強調表示の実装、および画像・動画による情報量の拡充。

- [x] **2.1 動的フローチャート強調 (Dynamic Highlighting)**
- [x] **2.2 画像・動画管理ギャラリー** (2026-02-05 実装)
- [x] **2.3 検索機能と関連表示** (2026-02-05 実装)

### Phase 3: 編集・バージョン管理・多人数対応 [完了]
**目標**: 運用現場での即時修正と、変更履歴の透明性を確保し、複数人での同時利用を安定させる。

- [x] **3.1 統合編集エクスペリエンス** (閲覧中のマニュアルを即座に編集開始)
- [x] **3.2 バージョン管理・履歴スナップショット** (変更前の状態を自動保存、ワンクリック復元)
- [x] **3.3 共有データ・排他制御**
    - [x] NAS等の共有フォルダ接続機能（DB/画像の同期）
    - [x] 複数人同時編集を防ぐ「ロック機能」
    - [x] 異常終了時の「強制ロック解除」機能
- [x] **3.4 データ保護と信頼性**
