# 診療所スタッフ向けマニュアル管理システム ユーザーマニュアル

## 1. はじめに
本システム「まゆみさん (MA-Assistant-Mayumisan)」は、診療所内の複雑な業務マニュアルを効率的に管理・閲覧するためのツールです。
特に、一つの業務手順（例：労災保険の対応）に対して、「受付」や「会計」など異なる業務視点からのアクセスを可能にし、それぞれの視点に必要な手順をフローチャートで強調表示します。

## 3. 基本的な使い方

### 3.1 マニュアルを探す
画面左側のサイドバーから、「フロー編集」「ガイド実行」「カテゴリ管理」を選択できます。
閲覧モード（ガイド実行）では、サイドバーから直接マニュアルを選択するか、カテゴリごとにフィルタリングされたリストから探すことができます。

### 3.2 マニュアルを作成する
1. **マニュアル作成画面**: 「フロー編集」タブを選択し、「ノード追加」や各種プロパティ設定を行います。
2. **基本情報の入力**: タイトルを入力します。
3. **手順の編集**: 各ノードのラベルやコメントを編集します。
4. **フローチャート**: React Flowを使用して視覚的にフローチャートを作成し、「保存」ボタンでデータベースに保存します。
5. **カテゴリ登録**: 「カテゴリ管理」画面で作成したカテゴリにマニュアルを紐付けます（Phase 1.5予定）。

### 3.3 マニュアルを閲覧する
- 「ガイド実行」からマニュアルを選択すると、手順をステップバイステップで進めることができます。
- 記述された作業指示（コメント）を確認しながら、次へのアクションを選択して進みます。

## 4. よくある質問 (FAQ)
- **Q: 画像は貼れますか？**
  - A: はい、ドラッグ＆ドロップで貼り付け可能です（Phase 3予定）。
- **Q: ネットワークがなくても使えますか？**
  - A: はい、本アプリはローカルPC内で完結して動作します。
