# 1. マニュアルの閲覧について

業務中に手順を確認したい場合の操作です。マニュアルを探す方法は主に2通りあります。

## 1.1 ホーム画面から探す
アプリを起動した直後の画面（ホーム）には、カテゴリやよく使うマニュアルが表示されています。

1.  **カテゴリカードの選択**:
    *   画面下部に並んでいる「受付」「会計」などの大きなカードをクリックすると、そのカテゴリが開きます。
2.  **マニュアルカードの選択**:
    *   カテゴリの中にあるマニュアルのカードをクリックすると、手順が表示されます。
3.  **パンくずリスト**:
    *   画面上部の「家アイコン ＞ 受付 ＞ ...」のような表示をクリックすると、上の階層に戻ることができます。

## 1.2 サイドバーから探す
画面左側のナビゲーションメニューからも操作できます。

1.  **サイドバーの開閉**:
    *   上部の「パネル」アイコンをクリックすると、サイドバーを折りたたんで画面を広く使えます。
2.  **カテゴリフォルダの開閉**:
    *   「＞」アイコンをクリックするとフォルダが開き、中身が表示されます。
3.  **マニュアルの選択**:
    *   目的のマニュアル名をクリックすると、メイン画面に表示されます。

## 1.3 全文検索機能（新機能）
ヘッダーにある検索窓から、すべてのマニュアルを対象としたキーワード検索が可能です。

1.  **キーワード入力**:
    *   検索窓に探したい言葉（手順名、本文、フローチャート内の言葉など）を入力します。
2.  **検索の実行**:
    *   **Enterキー**を押すと、検索が実行されます。
    *   マニュアル閲覧中に検索を実行した場合、自動的に検索結果一覧（ホーム画面）へ戻ります。
3.  **検索結果の確認**:
    *   ヒットしたマニュアルがカテゴリを跨いで一覧表示されます。
4.  **クリア**:
    *   検索窓の「×」ボタンを押すと入力がクリアされ、元のカテゴリ表示に戻ります。

## 1.4 閲覧モード
*   **ステップ表示**: 手順を1つずつ進めるモードです。大きな画像と分かりやすい説明で、作業ミスを防げます。
*   **レイアウト**: 画像がある場合は自動的に2カラム（説明と画像）に切り替わり、見やすさが最適化されます。

## 1.5 お気に入り登録
頻繁に使うマニュアルは、ホーム画面の上部に固定表示できます。
*   マニュアルタイトルの横にある**「☆（星）」アイコン**をクリックして、黄色（★）にすると登録されます。
