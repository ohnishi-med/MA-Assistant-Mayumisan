# 1. マニュアルの閲覧について

業務中に手順を確認したい場合の操作です。マニュアルを探す方法は主に2通りあります。

## 1.1 ホーム画面から探す
アプリを起動した直後の画面（ホーム）には、カテゴリやよく使うマニュアルが表示されています。

1.  **カテゴリカードの選択**:
    *   画面下部に並んでいる「受付」「会計」などの大きなカードをクリックすると、そのカテゴリが開きます。
2.  **マニュアルカードの選択**:
    *   カテゴリの中にあるマニュアルのカードをクリックすると、手順が表示されます。
3.  **パンくずリスト**:
    *   画面上部の「家アイコン ＞ 受付 ＞ ...」のような表示をクリックすると、上の階層に戻ることができます。

## 1.2 サイドバーから探す
画面左側のナビゲーションメニューからも操作できます。他の作業をしていても常に表示されているため便利です。

1.  **カテゴリフォルダの開閉**:
    *   「＞」アイコンをクリックするとフォルダが開き、中身が表示されます。
    *   「∨」アイコンをクリックするとフォルダが閉じます。
2.  **マニュアルの選択**:
    *   目的のマニュアル名をクリックすると、メイン画面に表示されます。

## 1.3 閲覧モードの種類
*   **一覧モード**: 全体の流れをスクロールして確認できます（標準）。
*   **対話モード**: 手順を1つずつ進めるモードです。「次へ」ボタンで進むため、作業ミスを防げます。画面左上の「ガイド実行」タブで切り替えます。

## 1.4 お気に入り登録
頻繁に使うマニュアルは、ホーム画面の上部に固定表示できます。
*   マニュアルタイトルの横にある**「☆（星）」アイコン**をクリックして、黄色（★）にすると登録されます。
