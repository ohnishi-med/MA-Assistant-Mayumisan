# 2. マニュアルの編集について

新しい業務手順を作成したり、既存の内容を修正する方法です。

## 2.1 新規作成
1.  左側のナビゲーションで、作成したいカテゴリ（フォルダ）を選択します。
2.  **「＋ 新しいマニュアル」**ボタン、またはナビゲーション上部の**「＋」**アイコンをクリックします。
3.  自動的に「手順1〜3」が入った状態で新規マニュアルが作成されます。

## 2.2 編集モードへの切り替え
1.  編集したいマニュアルを開きます。
2.  画面右上にある**「編集」ボタン**（鉛筆のアイコン）をクリックします。

## 2.3 内容の編集
*   **タイトルの変更**: 画面上部のタイトル文字をクリックして書き換えます。
*   **手順の入力**: 「手順1」などの見出しや、その下の説明文をクリックして編集します。
*   **表（テーブル）の挿入**: ステップ内の「テーブル追加」ボタン（表アイコン）で行います。
*   **画像の添付**:
    *   **クリップボードから**: 画像をコピーして、**Ctrl + V** で貼り付けます。
    *   **ファイル選択**: カメラアイコンをクリックして画像ファイルを選びます。

## 2.4 ステップの追加・削除
ステップの枠内を**右クリック**すると、以下のメニューが使えます。
*   **この上に追加**: 直前に新しいステップを挿入
*   **この下に追加**: 直後に新しいステップを挿入
*   **削除**: ステップを削除（元に戻せないので注意）

## 2.5 保存
編集後は必ず右上の**「保存」ボタン**（緑のチェック）を押してください。**「×」ボタン**を押すと、保存せずに編集をキャンセルします。

## 2.6 カテゴリの管理
マニュアルを整理するためのカテゴリ（フォルダ）を管理します。

*   **作成・名前変更**: サイドバーのカテゴリを右クリックすると、新しいフォルダの作成や名前の変更ができます。
*   **アイコン変更・並び替え**: 画面右上の設定アイコン（歯車）＞「カテゴリ管理」タブから、アイコンの変更やドラッグ＆ドロップによる並び替えが可能です。

## 2.7 未分類マニュアルの削除
「未分類」フォルダにあるマニュアルは、リスト上でマウスオーバーすると表示される「×」ボタンをクリックすることで削除できます。削除前には確認メッセージが表示されます。
