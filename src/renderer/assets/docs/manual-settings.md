# 3. 設定・バックアップについて

データの保存先変更やバックアップは、画面右上の**「設定」ボタン**（歯車アイコン）から行います。

## 3.1 保存先フォルダの設定
マニュアルデータの保存場所を指定します。
*   **「フォルダを選択する」**: PC内やNAS（共有サーバー）のフォルダを指定できます。共有フォルダに設定すれば、他のスタッフとマニュアルを共有できます。

## 3.2 自動保存
*   スイッチをONにすると、編集内容が自動的に保存されます。

## 3.3 手動バックアップ
*   **「今すぐ保存」**: 現在の全データを1つのファイル（.db）として書き出します。
*   **「データを読み込む」**: バックアップファイルを読み込んで、データを復元します（※現在のデータは上書きされるので注意してください）。
